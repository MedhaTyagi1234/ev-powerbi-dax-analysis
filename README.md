# 🚗 Dashboard: EV Performance Benchmarking

## 📊 Overview
This dashboard analyzes Electric Vehicle (EV) performance across brands, segments, and drivetrains.

The objective is to evaluate key factors such as driving range, battery capacity, drivetrain type, and charging performance.

## 🎯 Objectives
- Identify top EV brands based on driving range  
- Compare drivetrain performance (AWD, FWD, RWD)  
- Analyze battery capacity across different EV segments  
- Examine the relationship between fast charging power and driving range  

## 📌 Key Insights
- Luxury segment EVs show higher battery capacity  
- AWD vehicles dominate high-performance EVs  
- Higher fast charging power is linked with longer range  
- Driving range varies significantly across brands  

## 📈 Dashboard Features
- KPI cards for performance overview  
- Brand-wise range comparison  
- Drivetrain analysis  
- Segment-wise battery insights  
- Scatter plot: Charging Power vs Range  
- Interactive slicers  

## 🛠 Tools & Technologies
- Power BI  
- Data Modeling  
- DAX (Data Analysis Expressions)  

## 📌 Conclusion
The analysis shows how battery capacity, drivetrain type, and charging capability influence EV performance.

These factors help in identifying high-performing segments and comparing EV models effectively.
