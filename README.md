## ⚡ EV Power BI Analysis (DAX)

## 🧠 Objective
This project focuses on analyzing Electric Vehicle data using Power BI with a strong emphasis on DAX for advanced calculations and data modeling.
## 📊 Approach
The dataset is transformed and modeled in Power BI, where DAX is used to create calculated measures, KPIs, and dynamic insights.
## 🔹 Key Focus
- DAX-based calculations  
- KPI creation  
- Data modeling  
- Interactive dashboard development  
## 📌 Goal
To extract meaningful insights from EV data using powerful DAX functions and build a dynamic, analysis-driven dashboard.
